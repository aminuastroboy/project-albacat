# Project ALBACAT v4

Streamlit-based Islamic history audio platform for **Albani Zaria**.

## New in v4
- Home dashboard with continue listening
- Browse catalog with search, filters, and sorting
- Series playlist page
- Now Playing page with previous/next controls
- Transcript and summary page
- Analytics dashboard
- Research & Development Team page
- Admin guide
- Editable `catalog.json`, `team.json`, and `sample_catalog.json`

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Files to edit
- `catalog.json` for lectures
- `team.json` for research and development team information
- `sample_catalog.json` as a template for adding more lectures

## Logo
Place your logo image as:
- `logo.png`, or
- `assets/logo.png`

## Deployment
Push to GitHub and deploy with Streamlit Community Cloud.
