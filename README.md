# Project ALBACAT — Next.js + Tailwind v6

A production-style Next.js App Router build of Project ALBACAT with an HTML5 audio player.

## Setup
```bash
npm install
npm run dev
```

## Deploy
Push to GitHub and import the repo into Vercel.

## Public assets
Put your branding files here:
- `public/logo.png`
- `public/favicon.ico`
